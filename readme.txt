++++++++++++++++++++++++++++Buritto King Restaurant App++++++++++++++++++++++++++++++++++++

The application is created in Java.

Total Class: 8, including main class

Main class: Has all the commandline prompt available, commandline view can be changed from here.

All the food class: The details of all the food item is available in each food item named class, such as Burrito, Fries, Soda

Restaurant Manager Class: It is mainly the admin class, which basically controls all other class

Sales Report Class: To track down the overall sale per transaction sales report class initiated, it has all the method
regarding storing values, calculating total.

Preparation Time class: It is used to track down the preparation time that is being need for the orders.
